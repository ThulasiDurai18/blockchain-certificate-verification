# Blockchain-Based Certificate Verification System

## Overview
A secure system for issuing and verifying academic certificates using blockchain technology.

## Features
- Tamper-proof certificate storage using SHA-256 hashing
- Real-time verification
- Role-based access (Admin, Student, Verifier)
- RSA encryption for security

## Tech Stack
- Python (Flask, Streamlit)
- Blockchain concepts
- SHA-256, RSA
- Cloud Storage

## How to Run
```bash
pip install -r requirements.txt
python app.py
